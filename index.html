T-OLP 主要功能

T-OLP 根据提供的功能提供 Standard、Pro、Multi 选项。
而且，为了方便使用，另外提供JOB创建相关的办公模式和
用于生产线管理的生产线模式的UI。

Point 1.

创建简介的JOB文件

可以导入多种文件，无需逐一输入贴装位置信息、元件信息，即可轻松创建新的JOB文件。

模式 : Office
Point 2.

通过最优化JOB文件，确保最大产量

可以创建贴装顺序及供料器配置等最适合设备的JOB文件。
而且，通过生产线平衡最优化Cycle Time。

选项 : Standard / Pro / Multi
模式 : Office
Point 3.

Multi JOB文件创建

最优化类似型号的供料器配置、贴装顺序等，创建一个Multi JOB文件，因减少JOB Change而损失的时间。

选项 : Standard / Pro / Multi
模式 : Office
Point 4.

制定最佳的生产计划

可以对多个JOB文件制定最佳的生产计划。

选项 : Pro / Multi
模式 : Office
Point 5.

在生产过程中修改PCB文件
及预约生产&自动更改机型

可以在连接到生产线上的 T-OLP 上重新配置生产中的型号的吸嘴及供料器，或者修改贴装位置信息。

选项 : Standard / Pro / Multi
模式 : Line

通过 Multi 选项减轻投资负担

JOB最优化 : 可以使用贴装顺序、供料器配置、吸嘴自动配置，最优化JOB文件并提高生产效率。

重新优化 :对于现有的PCB程序和类似型号，可以使用与之前相同的供料器及吸嘴配置进行重新优化。这样做可以减少供料器及吸嘴更换时间。

元件平均分割 : 避免因特定元件的贴装位置过多导致负载集中在一个设备上。考虑供料器数量，自动移动贴装位置及调整生产线平衡。

前后相同的供料器配置 : 对接台车无需区分前后面，均可使用，并提高了生产效率。

打开基本生产型号的JOB文件后，连接与基本型号类似的其他型号的JOB文件，以便能够最快生产出各种品种。

可以对类似的生产型号统一应用供料器配置，减少换模时间，提高工作效率。

可以通过检查各种生产型号间的元件共享率，将两个型号之间的元件共享率高的型号分组，以便在生产属于该组的生产型号时无需更换供料器即可生产型号。

* 根据情况配置供料器时，还提供可以只对特定编号机型进行不同设置的“供料器Split”功能，提高工作效率。

可以最多注册100个JOB文件来执行生产计划，以便能最快生产出多个型号。

考虑总元件数量、班次及休息时间、 供料器/吸嘴/顶针更换时间 ，最优化生产顺序。

而且，可以预测生产时间及输出生产计划表，进行作业管理。

可以修改设备上已经打开的PCB程序的贴装位置、吸嘴配置、供料器配置，并与相关设备同步。

而且可以应用生产预约及自动更改机型功能，最大限度地提高生产效率。
生产预约功能 : 可以通过提前预约将要更改机型的型号，防止机型更改型号选择失误，并将机型更改自然延续到下一型号，最大限度地减少总生产时间及设备Idle时间。
自动更改机型功能 : 无需操作人员干预，可以自动打开设备的PCB文件，增加操作人员的便利性并缩短机型更改时间。
